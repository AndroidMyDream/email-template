# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2024-12-22

### Added
- ✨ 初始版本发布
- ✨ 支持三种邮件模板：欢迎邮件、重置密码、邮箱验证
- ✨ 多语言支持（中文/英文）
- ✨ TypeScript 类型定义
- ✨ React Email 组件集成
- ✨ Supabase 数据库表结构
- ✨ 示例 Edge Function 代码
- 📚 完整的 README 文档

### Features
- WelcomeEmail 组件：注册后的欢迎邮件
- ResetPasswordEmail 组件：密码重置邮件
- VerifyEmail 组件：邮箱验证邮件
- 响应式设计，兼容主流邮件客户端
- 可自定义 Logo、公司名称、支持邮箱等
